# Chocolate_sales_data
1.Title :
Chocolate Sales Performance Dashboard - A interactive data visualization tool built to show complete view of sales trends, region,sum of items sold and revenue generated. It shows items sold by sales person and amount generated by considering date .

2.Description/Purpose :
The purpose of this dashboard is to support sales managers and decision-makers in understanding how chocolate sales are performing over time. By visualizing sales by date, salesperson output, country revenue, and month-wise trends, it helps identify growth opportunities, track sales efficiency, and measure the effectiveness of sales operations. 

3.Tech stack :
The dashboard was built using the following tools and technologies:
Power BI Desktop : Main data visualization platform used for report creation.
Power Query : Data transformation and cleaning layer for reshaping and preparing the data.
DAX (Data Analysis Expressions) : Used for calculated measures, dynamic visuals, and conditional logic.
